# Финальная версия

В этой версии каталог и главная страница читают товары из PostgreSQL, а не из статичного массива.
Поэтому изменения в админке сразу становятся видны пользователям после обновления `/catalog`.

## Запуск

```bash
npm install
npx prisma generate
npm run dev
```

Если база пустая:

```bash
createdb krookies
npx prisma migrate dev --name init
npx prisma db seed
npm run dev
```

## Вход

Админ:

```text
+79959178862
1111
```

Клиент:

```text
любой российский номер
1111
```

## Если товар не виден в каталоге

Проверь в админке:

- `Опубликовано` = включено
- `В наличии` = включено

После сохранения товар должен появиться на `/catalog` после обновления страницы.


## Admin catalog

Добавлена страница `/admin/catalog`.

В админской шапке появилась кнопка `Каталог`. Она открывает клиентский каталог внутри админки: админская шапка сохраняется, а содержимое полностью повторяет страницу каталога с актуальными товарами из базы.
