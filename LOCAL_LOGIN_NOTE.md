# Локальный вход

В этой версии вход для локальной разработки работает как раньше:

- админ: `+79959178862`
- код: `1111`

Для обычного клиента можно использовать любой российский номер и код `1111`.

Поле `passwordHash` убрано из Prisma-схемы, поэтому не нужно вручную добавлять колонку в базу.
Если база уже создана, достаточно выполнить:

```bash
npx prisma generate
npm run dev
```

Если запускаете проект впервые:

```bash
createdb krookies
npx prisma migrate dev --name init
npx prisma db seed
npm run dev
```
