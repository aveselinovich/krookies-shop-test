# Stage 26 changes

- Добавлена обязательная проверка имени после авторизации: клиент без имени уходит на `/account/profile?required=name`.
- `/account/profile` остается доступен без имени, чтобы пользователь мог заполнить данные.
- Добавлена кнопка «Добавить товар» в `/admin/products`.
- Добавлена страница `/admin/products/new`.
- Добавлен API `POST /api/admin/products`.
- Slug создается автоматически и уникально на основе названия товара.
- В таблицу товаров возвращена колонка «Опубликовано».
- После создания товара пользователь возвращается в список товаров, после редактирования страница обновляется.
